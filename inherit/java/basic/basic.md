## Java中基本数据类型的存储方式和相关内存的处理方式



**1. java 是如何管理内存的**

	java 的内存管理就是对象的分配和释放问题。
	
**分配：**内存的分配是由程序完成的，需要new（基础类型除外），所有的对象在Heap中分配

**释放：**对象的释放是由垃圾回收机制决定和执行的，这样确实很方便，但是加重了JVM的工作。因为GC为了能够正确的释放对象，GC必须监控每一个对象的状态，包括对象的申请，引用，被引用，赋值等，GC都需要进行监控。
